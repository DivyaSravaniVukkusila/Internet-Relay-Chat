Compile Commands:
	 javac Server.javac
	 javac Client.java
Run Commands:
	 java Server
	 java Client
	
Commands for Server operations:
	1. Server quit:
		quit

Commands for Client operations:
	1. Create room:
		create <ROOMNAME/>
	2. Join room:
		join <ROOMNAME/>
	3. Listmembers room:
		listmembers <ROOMNAME/>
	4. Listrooms:
		listrooms	
	5. Groupmessage room:
		group <ROOMNAME/> <MESSAGE/>
	6. LeaveRoom room:
		leaveroom <ROOMNAME/>
	7. Privatemessge client:
		private <CLIENTNAME/> <MESSAGE/>
	8. Client quit:
		quit
